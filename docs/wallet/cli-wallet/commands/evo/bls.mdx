# bls

_bls "command" ..._

Set of commands to execute BLS related actions.

```bash
# Available commands:
  generate          - Create a BLS secret/public key pair
  fromsecret        - Parse a BLS secret key and return the secret/public key pair

```

`yerbas-cli bls generate`
