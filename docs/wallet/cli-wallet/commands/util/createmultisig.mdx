# createmultisig

_createmultisig nrequired ["key",...]_

Creates a multi-signature address with n signature of m keys required.
It returns a json object with the address and redeemScript.

```bash
# Create a multisig address from 2 addresses
yerbas-cli createmultisig 2 "[\"Xt4qk9uKvQYAonVGSZNXqxeDmtjaEWgfrs\",\"XoSoWQkpgLpppPoyyzbUFh1fq2RBvW6UK1\"]"
```

## Arguments

| Argument  | Type     | Description                     |
| --------- | -------- | ------------------------------- |
| "address" | (string) | The base58check encoded address |

```bash
Arguments:
1. nrequired      (numeric, required) The number of required signatures out of the n keys or addresses.
2. "keys"       (string, required) A json array of keys which are yerbas addresses or hex-encoded public keys
     [
       "key"    (string) yerbas address or hex-encoded public key
       ,...
     ]
```

## Output

| Result | Type | Description |
| ------ | ---- | ----------- |
| ...    | ...  | ...         |

```bash
Result:
{
  "address":"multisigaddress",  (string) The value of the new multisig address.
  "redeemScript":"script"       (string) The string value of the hex-encoded redemption script.
}
```
