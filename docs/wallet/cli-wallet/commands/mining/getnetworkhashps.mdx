# getnetworkhashps

_getnetworkhashps ( nblocks height )_

Returns the estimated network hashes per second based on the last n blocks.
Pass in blocks to override # of blocks, -1 specifies since last difficulty change.
Pass in height to estimate the network speed at the time when a certain block was found.

`yerbas-cli getnetworkhashps`

## Arguments

| Argument | Type      | Description                                                          |
| -------- | --------- | -------------------------------------------------------------------- |
| nblocks  | (numeric) | The number of blocks, or -1 for blocks since last difficulty change. |
| height   | (numeric) | To estimate at the time of the given height.                         |

```bash

Arguments:
1. nblocks     (numeric, optional, default=120) The number of blocks, or -1 for blocks since last difficulty change.
2. height      (numeric, optional, default=-1) To estimate at the time of the given height.
```

## Output

| Result | Type      | Description                 |
| ------ | --------- | --------------------------- |
| x      | (numeric) | Hashes per second estimated |

```bash
Result:
x             (numeric) Hashes per second estimated
```
