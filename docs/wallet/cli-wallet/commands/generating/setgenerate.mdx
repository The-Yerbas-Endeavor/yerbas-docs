# setgenerate

_setgenerate generate ( genproclimit )_

Set 'generate' true or false to turn generation on or off.
Generation is limited to 'genproclimit' processors, -1 is unlimited.
See the getgenerate call for the current setting.

```bash
# Set the generation on with a limit of one processor
yerbas-cli setgenerate true 1

# Check the setting
yerbas-cli getgenerate

# Turn off generation
yerbas-cli setgenerate false
```

## Arguments

| Argument     | Type      | Description                                                                 |
| ------------ | --------- | --------------------------------------------------------------------------- |
| generate     | (boolean) | Set to true to turn on generation, false to turn off.                       |
| genproclimit | (number)  | Set the processor limit for when generation is on. Can be -1 for unlimited. |

```bash
Arguments:
1. generate         (boolean, required) Set to true to turn on generation, false to turn off.
2. genproclimit     (numeric, optional) Set the processor limit for when generation is on. Can be -1 for unlimited.
```
