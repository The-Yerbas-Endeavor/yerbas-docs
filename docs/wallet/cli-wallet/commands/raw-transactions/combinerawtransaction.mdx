# combinerawtransaction

_combinerawtransaction ["hexstring",...]_

Combine multiple partially signed transactions into one transaction.
The combined transaction may be another partially signed transaction or a
fully signed transaction.

`yerbas-cli combinerawtransaction '["myhex1", "myhex2", "myhex3"]'`

## Arguments

| Argument  | Type     | Description                     |
| --------- | -------- | ------------------------------- |
| "address" | (string) | The base58check encoded address |

```bash
Arguments:
1. "txs"         (string) A json array of hex strings of partially signed transactions
    [
      "hexstring"     (string) A transaction hash
      ,...
    ]
```

## Output

| Argument    | Type     | Description                                       |
| ----------- | -------- | ------------------------------------------------- |
| "hexstring" | (string) | The hex-encoded raw transaction with signature(s) |

```bash
Result:
"hex"            (string) The hex-encoded raw transaction with signature(s)
```

## Examples

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "combinerawtransaction", "params": ["hex1", "hex2"] }' -H 'content-type: text/plain;' http://127.0.0.1:9998/
```
