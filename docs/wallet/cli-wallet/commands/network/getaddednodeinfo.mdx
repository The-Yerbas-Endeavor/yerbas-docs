# getaddednodeinfo

_getaddednodeinfo ( "node" )_

Returns information about the given added node, or all added nodes
(note that onetry addnodes are not listed here)

```bash
yerbas-cli getaddednodeinfo
yerbas-cli getaddednodeinfo "192.168.0.201"
```

## Arguments

| Argument | Type     | Description                                                                                 |
| -------- | -------- | ------------------------------------------------------------------------------------------- |
| "node"   | (string) | If provided, return information about this specific node, otherwise all nodes are returned. |

```bash
Arguments:
1. "node"   (string, optional) If provided, return information about this specific node, otherwise all nodes are returned.
```

## Output

| Result      | Type     | Description                                         |
| ----------- | -------- | --------------------------------------------------- |
| "addednode" | (string) | he node IP address or name (as provided to addnode) |

```bash
Result:
[
  {
    "addednode" : "192.168.0.201",   (string) The node IP address or name (as provided to addnode)
    "connected" : true|false,          (boolean) If connected
    "addresses" : [                    (list of objects) Only when connected = true
       {
         "address" : "192.168.0.201:9999",  (string) The yerbas server IP and port we're connected to
         "connected" : "outbound"           (string) connection, inbound or outbound
       }
     ]
  }
  ,...
]
```

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getaddednodeinfo", "params": ["192.168.0.201"] }' -H 'content-type: text/plain;' http://127.0.0.1:9998/
