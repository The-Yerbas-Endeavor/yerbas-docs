# disconnectnode

_disconnectnode "address" nodeid_

Immediately disconnects from the specified peer node.

Strictly one out of 'address' and 'nodeid' can be provided to identify the node.

To disconnect by nodeid, either set 'address' to the empty string, or call using the named 'nodeid' argument only.

```bash
yerbas-cli disconnectnode "192.168.0.6:9999"
yerbas-cli disconnectnode "" 1
```

## Arguments

| Argument  | Type     | Description                                         |
| --------- | -------- | --------------------------------------------------- |
| "address" | (string) | (string) The IP address/port of the node            |
| "nodeid"  | (string) | (string) The node ID (see getpeerinfo for node IDs) |
