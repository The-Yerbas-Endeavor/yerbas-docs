# walletpassphrasechange

_walletpassphrasechange "oldpassphrase" "newpassphrase"_

Changes the wallet passphrase from 'oldpassphrase' to 'newpassphrase'.

```bash
yerbas-cli walletpassphrasechange "old one" "new one"
```

## Arguments

| Argument        | Type     | Description            |
| --------------- | -------- | ---------------------- |
| "oldpassphrase" | (string) | The current passphrase |
| "newpassphrase" | (string) | The new passphrase     |

