# importwallet

_importwallet "filename"_

Imports keys from a wallet dump file (see dumpwallet). Requires a new wallet backup to include imported keys.

```bash
# Dump the wallet
yerbas-cli dumpwallet "test"

# Import the wallet
yerbas-cli importwallet "test"
```
