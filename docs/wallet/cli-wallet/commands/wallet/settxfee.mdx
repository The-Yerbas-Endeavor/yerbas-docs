# settxfee

_settxfee amount_

Set the transaction fee per kB. Overwrites the paytxfee parameter.

`yerbas-cli settxfee 0.00001`

## Arguments

| Argument | Type      | Description                   |
| -------- | --------- | ----------------------------- |
| amount   | (numeric) | The transaction fee in YERB/kB |

```bash
Arguments:
1. amount         (numeric or string, required) The transaction fee in YERB/kB
```

## Output

| Result | Type  | Description |
| ------ | ----- | ----------- | -------------------------- |
| true   | false | (boolean)   | Returns true if successful |

```bash
Result:
true|false        (boolean) Returns true if successful
```

## Examples

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "settxfee", "params": [0.00001] }' -H 'content-type: text/plain;' http://127.0.0.1:9998/
```
