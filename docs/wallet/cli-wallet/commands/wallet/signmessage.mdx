# signmessage

_signmessage "address" "message"_

Sign a message with the private key of an address

```bash
# Unlock the wallet for 30 seconds
yerbas-cli walletpassphrase "mypassphrase" 30

# Create the signature
yerbas-cli signmessage "XwnLY9Tf7Zsef8gMGL2fhWA9ZmMjt4KPwG" "my message"

# Verify the signature
yerbas-cli verifymessage "XwnLY9Tf7Zsef8gMGL2fhWA9ZmMjt4KPwG" "signature" "my message"
```

## Arguments

| Argument  | Type     | Description                                       |
| --------- | -------- | ------------------------------------------------- |
| "address" | (string) | The yerbas address to use for the private key. |
| "message" | (string) | The message to create a signature of.             |

```bash
Arguments:
1. "address"         (string, required) The yerbas address to use for the private key.
2. "message"         (string, required) The message to create a signature of.
```

## Output

| Result      | Type     | Description                                     |
| ----------- | -------- | ----------------------------------------------- |
| "signature" | (string) | The signature of the message encoded in base 64 |

```bash
Result:
"signature"          (string) The signature of the message encoded in base 64
```
