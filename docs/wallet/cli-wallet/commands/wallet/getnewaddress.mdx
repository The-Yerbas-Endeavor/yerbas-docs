# getnewaddress

_getnewaddress ( "account" )_

Returns a new Yerbas address for receiving payments.
If 'account' is specified (DEPRECATED), it is added to the address book
so payments received with the address will be credited to 'account'.

`yerbas-cli getnewaddress`

## Arguments

| Argument  | Type     | Description                                                                                                                                                                                                                                                                     |
| --------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| "account" | (string) | The account name for the address to be linked to. If not provided, the default account "" is used. It can also be set to the empty string "" to represent the default account. The account does not need to exist, it will be created if there is no account by the given name. |

```bash
Arguments:
1. "account"        (string, optional) DEPRECATED. The account name for the address to be linked to. If not provided, the default account "" is used. It can also be set to the empty string "" to represent the default account. The account does not need to exist, it will be created if there is no account by the given name.
```

## Output

| Result    | Type     | Description               |
| --------- | -------- | ------------------------- |
| "address" | (string) | The new yerbas address |

```bash
Result:
"address"    (string) The new yerbas address
```
