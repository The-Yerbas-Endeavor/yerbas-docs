# dumpprivkey

_dumpprivkey "address"_

Reveals the private key corresponding to 'address'.
Then the importprivkey can be used with this output

```bash
# Examples:
yerbas-cli dumpprivkey "myaddress"
yerbas-cli importprivkey "mykey"
```

## Arguments

| Argument  | Type     | Description                               |
| --------- | -------- | ----------------------------------------- |
| "address" | (string) | The yerbas address for the private key |

```bash
Arguments:
1. "address"   (string, required) The yerbas address for the private key
```

## Output

| Result | Type     | Description     |
| ------ | -------- | --------------- |
| "key"  | (string) | The private key |

```bash
Result:
"key"                (string) The private key
```

## Examples

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "dumpprivkey", "params": ["myaddress"] }' -H 'content-type: text/plain;' http://127.0.0.1:9998/
```
