# getchaintxstats

_getchaintxstats ( nblocks blockhash )_

Compute statistics about the total number and rate of transactions in the chain.

`yerbas-cli getchaintxstats `

## Arguments

| Argument    | Type      | Description                                                  |
| ----------- | --------- | ------------------------------------------------------------ |
| nblocks     | (numeric) | Size of the window in number of blocks (default: one month). |
| "blockhash" | (string)  | The hash of the block that ends the window.                  |

```bash
Arguments:
1. nblocks      (numeric, optional) Size of the window in number of blocks (default: one month).
2. "blockhash"  (string, optional) The hash of the block that ends the window.
```

## Output

| Result    | Type      | Description                                                     |
| --------- | --------- | --------------------------------------------------------------- |
| "time"    | (numeric) | The timestamp for the statistics in UNIX format.                |
| "txcount" | (numeric) | The total number of transactions in the chain up to that point. |
| "txrate"  | (numeric) | The average rate of transactions per second in the window.      |

```bash
Result:
{
  "time": xxxxx,        (numeric) The timestamp for the statistics in UNIX format.
  "txcount": xxxxx,     (numeric) The total number of transactions in the chain up to that point.
  "txrate": x.xx,       (numeric) The average rate of transactions per second in the window.
}
```
