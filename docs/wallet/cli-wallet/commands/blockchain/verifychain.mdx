# verifychain

_verifychain ( checklevel nblocks )_

Verifies blockchain database.

`yerbas-cli verifychain `

## Arguments

| Argument   | Type      | Description                             |
| ---------- | --------- | --------------------------------------- |
| checklevel | (numeric) | How thorough the block verification is. |
| nblocks    | (numeric) | The number of blocks to check.          |

```bash
Arguments:
1. checklevel   (numeric, optional, 0-4, default=3) How thorough the block verification is.
2. nblocks      (numeric, optional, default=50, 0=all) The number of blocks to check.
```

## Output

| Result | Type  | Description |
| ------ | ----- | ----------- | --------------- |
| true   | false | (boolean)   | Verified or not |

```bash
Result:
true|false       (boolean) Verified or not
```
