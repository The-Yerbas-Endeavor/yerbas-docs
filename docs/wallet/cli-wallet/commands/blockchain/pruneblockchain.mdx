# pruneblockchain

_pruneblockchain_

`yerbas-cli pruneblockchain 1000`

<!-- <ReactPlayer playing controls url={[

## Arguments

| Argument | Type     | Description                                                                                                                                                                   |
| -------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| "height" | (string) | The block height to prune up to. May be set to a discrete height, or a unix timestamp to prune blocks whose block time is at least 2 hours older than the provided timestamp. |

```bash
Arguments:
1. "height"       (numeric, required) The block height to prune up to. May be set to a discrete height, or a unix timestamp
                  to prune blocks whose block time is at least 2 hours older than the provided timestamp.
```

## Output

| Result | Type      | Description                      |
| ------ | --------- | -------------------------------- |
| n      | (numeric) | Height of the last block pruned. |

```bash
Result:
n    (numeric) Height of the last block pruned.
```

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "pruneblockchain", "params": [1000] }' -H 'content-type: text/plain;' http://127.0.0.1:9998/
