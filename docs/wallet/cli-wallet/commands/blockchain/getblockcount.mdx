# getblockcount

Returns the number of blocks in the longest blockchain.

`yerbas-cli getblockcount `

## Output

| Result | Type     | Description             |
| ------ | -------- | ----------------------- |
| "n"    | (number) | The current block count |

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getblockcount", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:9998/
```
