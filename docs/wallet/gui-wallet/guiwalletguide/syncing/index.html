Syncing

Syncing

If this is the first time running the YERB wallet,it may take several minutes to an hour to sync. Sync status will appear in the bottom left of the window.

note

The wallet must sync fully to the current network block height before you can see your current balance or send funds from it! You can always check the network's current block height at the Yerbas Explorer
